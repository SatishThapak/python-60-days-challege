# 🐍 Introduction to Python

## What is Python?
Python is a popular, high-level programming language created by **Guido van Rossum** and first released in **1991**.  
It is widely used for:

- Web development (server-side)
- Software development
- Mathematics and data analysis
- System scripting

---

## What can Python do?
- Run on a server to create web applications  
- Integrate with software to build workflows  
- Connect to databases, read, and modify files  
- Handle big data and perform complex mathematics  
- Support rapid prototyping and production-ready applications  

---

## Why Python?
- Works across multiple platforms: **Windows, Mac, Linux, Raspberry Pi**  
- Simple syntax, similar to English  
- Requires fewer lines of code compared to other languages  
- Runs on an interpreter system → quick prototyping  
- Supports **procedural**, **object-oriented**, and **functional** programming styles  

---

## Good to Know
- The most recent major version is **Python 3**  
- Code can be written in a simple text editor or in an IDE such as **Thonny, PyCharm, NetBeans, or Eclipse**  
- IDEs are especially useful for managing larger projects  

---

## Python Syntax Compared to Other Languages
- Designed for **readability**, influenced by English and mathematics  
- Uses **new lines** to end commands (instead of semicolons or parentheses)  
- Relies on **indentation (whitespace)** to define scope (loops, functions, classes)  
- Other languages often use **curly brackets `{}`** for scope
